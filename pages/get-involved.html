{% trans "Get Involved" %}

{% trans "Software Developers" %}

{% blocktrans %}Mixxx is written in C++ and uses the Qt platform library for most data structures, graphics drawing, and window handling. Fortunately, Qt is easy to use and very well documented, so you don't need any prior Qt or audio programming experience to start hacking on Mixxx.{% endblocktrans %}

{% blocktrans %}Besides creating cool DJ software and working with a fun team, contributing to Mixxx also looks great on your resume or CV. It's also an ideal way to learn about real-time audio programming, Qt, or even just hone your C++ skills.{% endblocktrans %}

{% trans "Start Developing Now" %}

  • {% blocktrans %}Get a Github account and fork your own branch!{% endblocktrans %}

  • {% blocktrans %}Pick a bug off our Easy Bugs List and start hacking!

    Fixed it? Your name will be in the Mixxx credits!{% endblocktrans %}

  • {% blocktrans %}When you're stuck, get help from our existing developers our IRC channel: #mixxx on Freenode{% endblocktrans %}

  • {% blocktrans %}Join our mailing list: mixxx-devel{% endblocktrans %}

  • {% blocktrans %}You don't even have to know C++. There are developers who got into Mixxx development while learning C++ along the way.{% endblocktrans %}
